# Dota2-AutoAccept-Utility

------ENGLISH----------

A simple and efficient Python-based utility to automatically accept Dota 2 matches.

## 📌 Description
This tool is designed to automate the match-accepting process. It uses `pyautogui` to monitor the game interface and automatically click the "Accept" button as soon as a match is found.

## 🚀 Key Features
* Automatic detection of the match-accept button.
* Low system resource usage.
* Easily customizable for different screen resolutions.

## 🛠 Tech Stack
* **Language:** Python
* **Libraries:** `customtkinter` (GUI), `pyautogui` (automation), `opencv-python` (image processing).

## ⚙️ How to Run
1. Ensure you have Python (version 3.8+) installed.
2. Clone the repository or download the source files.
3. Install the required dependencies:
   ```bash
   pip install customtkinter pyautogui opencv-python
